# WeChat-LoveSim(恋语小助手)

**像玩galgame一样和你的好友增进感情。**

## 项目背景

你的生活中是否有这样的情况：

* 好不容易加了个好友，结果聊天就只剩下“嗯”“哦”“哈哈”，整个聊天像在跟空气打招呼。
* 明明想刷好感、聊暧昧，却总是被冷场或者尬聊支配。
* 朋友圈别人晒恩爱，你在旁边默默划手机，心里默念：“聊天为什么这么难啊？”

没关系，你不是孤单一人。程序员的社交技能，有时候真的比敲代码还难。

于是我决定做一个 **WeChat-LoveSim（恋语小助手）**，把你的微信好友 **变成游戏里的攻略对象**。
你只需要选择三个选项中的一个：

* **A**：甜甜的夸一句 → 好感度 +1
* **B**：机智吐槽一下 → 氛围 +1
* **C**：奇怪操作搞怪一下 → 让对方怀疑你是不是疯了

就这样，你既能保持聊天互动，又能体验养成游戏的乐趣——每天都能和好友有点新鲜感。

> 总结一句：这是一个让你 **聊天轻松 + 氛围搞怪 + 心理平衡** 的小工具，比背聊天技巧强多了。
