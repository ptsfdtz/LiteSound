# LiteSound

一个基于 Tauri + React 的轻量级桌面音乐播放器。

## 项目简介

LiteSound 是一个现代化的桌面音乐播放器应用，采用 Tauri 2 + React 19 + TypeScript 技术栈构建。项目致力于提供简洁美观的用户界面和流畅的使用体验。

## 技术栈

- **前端**: React 19.2.0 + TypeScript 5.8.3 + Vite 7.2.4
- **桌面框架**: Tauri 2.9.4
- **样式**: Tailwind CSS 4.1.17 + CSS Modules
- **包管理**: pnpm

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 启动 Tauri 应用
pnpm tauri dev

# 构建应用
pnpm tauri build
```
